Design

The site has been made with design in mind. A flat graphical style makes for a pretty user interface. The site is weighted towards using the most standard CSS.

Speed

The site is simple and lightweight, it uses the least javascript possible. Making use of the optimized and efficient language of CSS to increase speed.
